Vector style files

Nutiteq vector maps uses Mapnik XML for styling. Following style files are compatible with Nutiteq vector Maps service data structure. Slow-performing styling is avoided, and some our extensions adaptations are included:e.g. 3D-specific and language parameters. So we suggest to use these as base for your own customized styles.

  • nutibright-v3.zip – Nutiteq Bright for general use. “Bright” is the default choice, same package includes also “nutiteq_dark” and “nutiteq_grey” (updated 25.04.2016)
