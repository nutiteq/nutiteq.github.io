Nutiteq Maps SDK

Developer site

Fork me on GitHub

Here you can find a developer projects for Nutiteq Maps SDK. Key features:

  • smooth map panning and zooming
  • vector overlays/layers
  • raster overlays/layers
  • 2.5/3D view: map tilting, rotating and 3D models
  • offline mapping (MBTiles, persistent caching etc)
  • supports any map projection, which makes it more usable for GIS apps
  • great performance and a lot of customization options

HelloMap3D for iOS (beta)

For iOS downloads and guides see Nutiteq HelloMap3D-iOS project Wiki pages. It shows how to integrate SDK with your app, create a map and add layers on top of that, and also advanced tasks. This is with Nutiteq Maps SDK 3.x which is in beta now.

Nutiteq Maps SDK for iOS (beta)

HelloMap3D for Android

For Android downloads and guides see Nutiteq HelloMap3D project Wiki pages. It has even more developer samples and guides. This is with Nutiteq Maps SDK 2.x (production version)

Contribution

We welcome any kind of contributions: questions, pull requests, questions, feedback, wiki updates etc. Please join nutiteq-dev@googlegroups.com for discussions. To subscribe to the group via email (which does not need to be gmail), send an email to nutiteq-dev+subscribe@googlegroups.com
