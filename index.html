Nutiteq Maps SDK

Sample app projects for developers

Nutiteq Maps SDK key features:

  • Unified native API for iOs and Android
  • Smooth map panning and zooming
  • Offline maps
  • Vector-based high-performance base map
  • Indoor map support
  • Vector and vector overlays/layers
  • 2.5/3D view: map tilting, rotating and 3D models and 3D city
  • Highly customizable: custom data sources / formats, projections, visual elements, styles etc

Nutiteq SDK 3.0 for iOS

iOS downloads and getting started guides are in Nutiteq HelloMap3D-iOS project. It shows how to integrate SDK with your app, create a map and add layers on top of that, and also some advanced tasks.
