Pro extensions

Nutiteq SDK Enterprise package enables to add options which are used for more advanced apps: for GIS, B2B, military, utility management and also in some end-user apps. Public SDK build does not include these features. Please contact sales@nutiteq.com to get information how to enable and license these features, and to get evaluation copy of the extensions.

GIS extension

  • Vector GIS layers: KML, Shapefile, MapInfo and many others
  • Raster GIS layers: GeoTIFF, BSB, ECW, MrSID etc
  • Editable vector layers
  • Custom projections

3D converter tool

  • Enables to create 3D layers with custom data
  • Convert Collada/DAE files to 3D models which can be put to map
  • Convert KMZ 3D city to special map layer
